# circuit-rc-decharge
Etude de la décharge d'un circuit RC avec une carte Arduino. Allure temporelle et étude statistique.
* le notebook [circuit_rc_decharge.ipynb](circuit_rc_decharge.ipynb) donne le circuit à réaliser et exploite les résultats de l'acquisition temporelle.
* le notebook [circuit_rc_auto.ipynb](circuit_rc_auto.ipynb) présente les résultats de mesures répétées du temps caractéristique.
* les programmes à charger dans l'Arduino sont dans les répertoires `circuit_rc_decharge` et `circuit_rc_auto`
* les progammes Python permettant de communiquer avec l'Arduino et d'enregistrer les données sont `circuit_rc_decharge_acquisition.py` et `circuit_rc_auto_acquisition.py`
